History of Web

The web has a great history and created a gigantic impact by connecting , sharing resources over the internet. It was invented by Tim Berners-Lee. The fundamental goal behind the WWW was to create a robust and user-friendly worldwide information system by combining and connecting webpages with the Hypertext protocols and utilisng hyperlinks for connecting with different web pages over the internet.

Hyperlinks

Hyperlinks are the most popular core functionality of web which connects with other pages over the internet. A hyperlink, sometimes called a link, is a feature on a web page that, with a single click or touch, enables users to move to another resource on the same website, another website, or inside the same web page. Usually, hyperlinks are shown as text, photos, or other media items that may be clicked. A hyperlink causes the associated resource—which might be any kind of digital content—to load when a user clicks on it. This resource could be another webpage, file, picture, video, or anything else. This fluidity of resource switching is what gives the web its immense strength and dynamic nature.

Implementation of Hyperlinks

Hyperlinks are implemented using anchor tag <a>. This tag makes the content clickable and can redirect to external pages or helps in internal routing. It requires certain attributes to be functional and below are the major attributes used with anchor tag.

Attibutes:

  • href: This attribute is mandatory to perform the action. This attribute will have a path to any external or internal resource. When the user clicks on this the routing is done to navigate to the link which is mentioned in this attribute.
  • target: This attribute is optional but it is used to tell the browser to navigate the link to a new tab or same tab.
  • title: This attribute is also optional and it is used to provide additional information as a tooltip.

Examples

A hyperlink with plain text.

<a href="www.google.com">google.com</a>

Output:

google.com

A hyperlink with target attribute which navigates to a new tab.

<a href="www.google.com" target="_blank">google.com</a>

Output:

google.com

Working of Hyperlinks

A remarkable interaction takes place between the browser and the server when a user clicks a hyperlink:

  1. User Click Action: The user clicks on hyperText or a button which is embedded with hyperlink.
  2. Browser Indentification: The Browser indentifies the user acrtion and computes the anchor tag which was clicked.
  3. HTTP Request: A request is made to the destination server where the embedded URL is targeting by the Browser.
  4. Server Reaction: The HTTP request is received and handled by the web server. The server gets the requested data based on the kind of resource (HTML page, picture, video, etc.).
  5. Response Data: The Server understands the requested data and sends the relevant appropriate response back to the browser.
  6. Browser Render Process: The Browser receives the reponse by the server and renders the page.
